【阿里云】【云起实验室】【才发现这么好的网站~】

今天的实验课老师推荐了一个平台：阿里云-云起实验室。这种模式与“大数据竞赛”的青椒课堂方式相似，也是我在2021年期待很久的一个比赛，后面因疫情没有参加线下比赛。这里资源很多，值得学习很久了。

2022/10/20实验：MySQL基础-学生管理系统数据库设计

1. 创建资源

单击屏幕右侧 创建资源 按钮，免费创建当前实验所需云产品资源。

说明：
资源创建过程需要1~3分钟。完成实验资源的创建后，您可以在 云产品资源 列表查看已创建的资源信息，例如：IP地址、子用户名称和子用户密码等。

2. 登录RDS控制台

  1. 点击右侧图标切换到远程桌面操作界面，双击打开桌面 FireFox浏览器，在RAM用户登录框中点击“下一步”，复制上面的“子用户密码” ，按CTRL + V把密码粘贴到密码输区登陆子账户（后续在远程桌面里的粘贴操作均使用CTRL + V快捷键）。

  2. 在阿里云管理控制台，点击顶部【产品与服务】 ，然后点击 【数据库】条目下的 【云数据库RDS版】 ，进入RDS数据库的管理页面。

  3. 在【实例列表】页面中，选择 云产品资源 中提供的 地域，然后查看已创建的RDS实例。


    3. 创建数据库账号

  4. 单击已创建的RDS实例ID，进入数据库管理页面。

  5. 在实例详情页，单击左侧导航栏的【账号管理】，然后单击【创建账号】。

  6. 参考说明配置账号信息，然后单击【确定】。

数据库账号：输入数据库账号名称，例如super_user。
账号类型：选择高权限账号。
密码：设置账号密码。
确认密码：再次输入密码。

4. 连接DMS控制台

  1. 在数据库管理页面，单击页面右上方的【登录数据库】。

  2. 在输入框中，输入刚才创建的用户名和密码，单击【登录】。

登录成功，进入数据管理服务系统DMS。


5. 创建数据库

  1. 在DMS控制台左侧实例列表，单击图标。

  2. 在DMS控制台左侧实例列表，右键单击【实例名称】，然后单击【数据库管理】。

  3. 在数据库管理页面，单击【创建库】。

  4. 输入数据库名stu_coutse_sc，字符集utf8，校验规则utf8_general_ci。


    6. 创建数据表

  5. 在左侧实例列表单击【刷新】按钮，然后双击切换数据库为为上一小节创建的stu_coutse_sc。

  6. 单击左侧“点击这里开始建表吧”，创建表。

  7. 创建student学生表。

体验：可视化创建表自动生成SQL语言执行

7. 编辑表关系

  1. 设置成绩表sc与课程表course之间、成绩表sc与学生表student之间的关系。

a. 右键单击表sc，然后单击【新开SQL窗口】。

b. 在SQL窗口中，执行以下SQL语句。

alter table sc add constraint fk_cid foreign key(cid) references course(cid) ;
alter table sc add constraint fk_sid foreign key(sid) references student(sid) ;

  1. 设置教师表teacher和课程表course之间的关系。

在SQL窗口中，执行以下SQL语句。

alter table course add constraint fk_tid foreign key(tid) references teacher(tid) ;

点击执行(F8)
